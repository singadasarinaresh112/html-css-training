Check if a string is a palindrome in Javascript

First exercice

The goal of this exercise is to create a function which can detect a palindrome. A palindrome is a word that can be read in two directions without changing its significance. Example: “kayak” can be read in both sides.
Optional: create one input in the HTML to allow users to enter any kind of word and then print the result in the DOM. Display an error if the content entered is not a string.
  • “kayak” = true
  • “radar” = true
  • “maman” = false
  • “Koalect” = false
  • “Bob” = true
